# Zepto-Sales-Customer-Analysis-SQL-Python-📊

## 📌 Project Overview

This project focuses on analyzing customer purchase behavior and product performance using SQL, Python, and Power BI. The goal is to extract meaningful insights that can help in business decision-making.

---

## 🎯 Objectives

* Analyze customer purchasing patterns
* Identify top-selling products
* Understand the impact of discounts on sales
* Explore category-wise performance

---

## 🛠️ Tools & Technologies

* SQL
* Python (Pandas, Matplotlib/Seaborn)
* Power BI

---

## 📂 Project Structure

* **data/** → Raw dataset (CSV file)
* **notebooks/** → Jupyter Notebook for data analysis
* **sql/** → SQL queries used for analysis
* **images/** → Visualizations (optional)

---

## 🔍 Key Insights

* Identified top-performing products contributing to overall revenue
* Observed how discounts influence customer purchase behavior
* Found category-wise trends and demand patterns
* Analyzed customer buying frequency and preferences

---

## 📊 Dashboard

An interactive Power BI dashboard was created to visualize key metrics like:

* Sales trends
* Product performance
* Category distribution

---

## 🚀 Learnings

* Improved SQL querying and data extraction skills
* Gained hands-on experience in data cleaning and analysis using Python
* Learned how to convert raw data into actionable insights

---

## 📎 How to Use

1. Download the dataset from the `data/` folder
2. Run SQL queries from the `sql/` folder
3. Open the notebook in Jupyter to see analysis and visualizations
